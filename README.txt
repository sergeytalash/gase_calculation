Описание файлов и папок программы Gase Calculator

run_on_windows.bat - Запуск программы

data - Папка, в которой находятся файлы измерений, файл концентраций и результаты вычислений
data/baloon_concentration.xlsx - Информация по концентрации газов
data/table_by_1_minute.xlsx - Промежуточные ежеминутные измерения
data/table_ch4.xlsx - Вывод значений CH4
data/table_co2.xlsx - Вывод значений CO2

gase_calculation - Папка, в которой находятся исходные копии файлов, загруженных из интернета

_update.bat - Обновляет программу из интернета

main.py - Исходный текст программы

requirements.txt - Список необходимых модули для работы программы (используется только при первичной установке программы)
